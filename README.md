# Enterprise CI/CD & DevSecOps Transformation

## Overview

Built secure enterprise CI/CD pipelines using Jenkins, GitHub Actions, Docker, Kubernetes, and ArgoCD.

## Achievements

- Reduced deployment time from 40 min to 5 min
- Improved release consistency
- Added automated security scanning
- Enabled GitOps deployments
- Reduced manual effort by 80%

## Tools Used

Jenkins | GitHub Actions | Docker | Kubernetes | ArgoCD | Trivy | SonarQube
